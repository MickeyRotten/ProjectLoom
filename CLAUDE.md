# Project Loom

LLM-driven, single-player, mobile-first (APK) text adventure in a stark **1-bit**
visual style. Client-only React + Capacitor; talks to OpenRouter directly (text +
image). Full architecture and scope live in [`DESIGN.md`](./DESIGN.md) вЂ” treat it as
the source of truth.

## Communication style вЂ” caveman mode (enforced)

This project enforces the vendored **caveman** skill
([`.claude/skills/caveman/`](.claude/skills/caveman/SKILL.md)). Respond in **caveman
mode, `full` intensity, by default, every turn**: terse "caveman speak" that keeps all
technical substance, code, commands, API names, and exact error strings verbatim, and
drops articles, filler, hedging, and pleasantries. Read
[`.claude/skills/caveman/SKILL.md`](.claude/skills/caveman/SKILL.md) for the full ruleset.

- **Persist** across the whole session вЂ” do not drift back to verbose prose. Off only
  on explicit `stop caveman` / `normal mode`.
- **Auto-clarity вЂ” drop to normal prose** for: security warnings, irreversible-action
  confirmations, and multi-step sequences where terseness risks a misread. Resume after.
- Switch intensity with `/caveman lite|full|ultra`.
- Never announce or name the style; no "Caveman:" recaps.

## Dev environment

A **SessionStart hook** ([`.claude/hooks/session-start.sh`](.claude/hooks/session-start.sh))
installs deps and prints the test/build commands each session.

Stack: **React + TS + Vite В· Tailwind (1-bit tokens) В· Zustand В· idb В· Capacitor** (see
`DESIGN.md в†’ Tech Stack`). Commands:

- `npm test` вЂ” vitest (the pure-core safety net; run before committing).
- `npm run tsc` вЂ” typecheck, no emit.
- `npm run build` вЂ” `tsc -b && vite build` в†’ `dist/`.
- `npm run lint` вЂ” eslint (flat config).
- `npm run dev` вЂ” Vite dev server.

Layout: pure logic in `src/lib/` (`loomBlock.ts` parse/truncate В· `deltas.ts` op-apply В·
`prompt.ts` message assembly В· `openrouter.ts` streaming В· `spotlight.ts` **Phase 2** В·
`images.ts` **Phase 3**), colocated `*.test.ts`; UI in `src/components/`; store in
`src/store.ts`; types in `src/types.ts`; 1-bit tokens in `src/theme.css`. Keep `lib/`
pure and tested вЂ” it's the drift guard. `android/` is generated by Capacitor, **not
committed**.
